## 安装部署

安装brew:
	
	$ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
	
通过brew安装mongodb:

	$brew install mongodb

安装成功:

![Mou icon](./images/mongodb2.jpg)

	$mongod --version
